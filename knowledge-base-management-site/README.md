# Knowledge Base Management Site

Standalone recreation of the Figma Knowledge Management Portal prototype.

## Scope

- Lessons Learned landing page
- Top navigation and breadcrumbs
- Search, filters, keyword chips, grid/list toggle
- Responsive lesson cards
- Create lesson dialog
- Floating AI assistant action

## UX Improvements Applied

- Semantic landmarks and skip link
- Keyboard-visible focus states
- Accessible color contrast
- Responsive mobile navigation
- Reduced visual clutter in filters
- Grid and list view for different scanning preferences
- Search shortcut with visible `Ctrl K` hint
- Viewport-safe dialog sizing

## Run

Open `index.html` directly in a browser.
