# HR Tool V2

HR Tool V2 is a React + Vite implementation of a modern HR management interface based on the “HR System Development” Figma design system, focusing on reusable components and clean, data-centric layouts for HR workflows.

Original Figma project: [https://www.figma.com/design/Z8lVZssRryEplcmvDfDnnQ/HR-System-Development](https://www.figma.com/make/Z8lVZssRryEplcmvDfDnnQ/HR-System-Development?code-node-id=0-9&p=f&t=ngG6ADGmKXc9nKCe-0&fullscreen=1)
